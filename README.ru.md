Автоматический скрипт на Python для сборки кастомной версии игрового движка **Godot Engine (4.7)** с поддержкой **C# (Mono)** на Windows и автоматическим шифрованием шаблонов экспорта.

## ✨ Основные возможности

* **Управление зависимостями:** Автоматически находит или предлагает установить `SCons`, а также опциональные SDK (Direct3D 12, AccessKit, WinRT).
* **Безопасность:** Автоматически генерирует 32-байтный AES-ключ для защиты скриптов проекта и встраивает его в релизный шаблон экспорта.
* На данный момент возможно создание релизных шаблонов только под Windows!

## 🛠 Требования к системе

Перед запуском скрипта убедитесь, что в вашей системе Windows установлены:
1. **Python 3.x** (обязательно добавленный в `PATH`).
2. **Git**.
3. **.NET SDK** (необходим для сборки модулей Mono/C#).
4. Установленный Visual Studio и компилятор C++ (обязательно добавленный в `PATH`).

## 🚀 Инструкция по запуску

1. Клонируйте этот репозиторий себе на компьютер.
2. Запустите главный скрипт: *python main.py*
3. Следуйте подсказкам в консоли (y/n) для выбора необходимых компонентов сборки.

## 💎 По завершению создания шаблона:

1. Вы получите 32-байтный AES-ключ и шаблон экспорта в папке bin.
2. В редакторе Godot перейдите: Проект -> Экспорт -> Шифрование.
3. Включите: Шифровать экспортированный PCK
4. Установите \*.* в поле: Фильтры для включения файлов/папок.
5. Вставьте сгенерированный ключ в поле: Ключ шифрования.
6. Перейдите в параметры экспорта и укажите путь до релизного шаблона экспорта.

## ⚙️ Какие шаги автоматизирует скрипт:
* Проверка установленных в системе git и scons.
* Клонирование ветки 4.7-stable из официального репозитория Godot.
* Сборка временного исполняемого файла редактора 4.7-stable.
* Генерация C#-файлов Glue.
* Сборка финальных управляющих библиотек GodotSharp.
* Сборка релизного шаблона экспорта с уникальным ключом шифрования.
